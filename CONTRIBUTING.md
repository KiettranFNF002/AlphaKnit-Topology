# Contributing to AlphaKnit

We welcome research-driven contributions!

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or research module.
3. Ensure all tests pass: `pytest tests/`.
4. Submit a Pull Request with a detailed description of your changes.

## Research Guidelines
AlphaKnit is a research-grade project. Please include ablation studies or metric comparisons if you are modifying the core generative architecture.
